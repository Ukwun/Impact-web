# School Onboarding and Cohort Management Workflow

## Onboarding Stages

1. School account provisioning
2. Admin and facilitator role assignment
3. Cohort setup by level and term window
4. Curriculum module activation
5. Learner enrollment import and parent linking
6. Timetable and live session scheduling
7. Monitoring kickoff (attendance and activation dashboards)

## Cohort Setup Inputs

- Programme: Senior Secondary or ImpactUni
- Term window: start and end dates
- Facilitator assignment
- Session cadence and replay policy
- Parent visibility policy (younger learners)

## Operational Controls

- Age-based access permissions
- Parent access flags for progress and attendance
- Moderation and code-of-conduct escalation path

## Notification Integrations

Reminder channels:
- WhatsApp
- Email
- SMS

Reminder events:
- Upcoming live session
- Assignment due
- Replay published
- Attendance alert below threshold

## School Admin Dashboard KPIs

- Activation: first-session attendance within 7 days
- Retention: weekly active learners per cohort
- Attendance: cohort attendance rate
- Completion: progression eligibility rate

# Assessment Rubrics and Badge Logic

## Assessment Components

- Quizzes and short tests
- Worksheets and practical tasks
- Reflection journals
- Project submissions
- Live attendance
- Participation and presentation scores

## Rule-Based Progression Thresholds

- Completion percentage >= 70
- Assessment average >= 60
- Project submission rate >= 60
- Live participation score >= 60

Progression is granted only when all thresholds are met.

## Participation Score Model

Weighted blend:
- Attendance minutes: 45%
- Quiz average: 30%
- Submission rate: 25%

## Recognition Mapping

- Skill badge: assessment average >= 85
- Attendance recognition: participation score >= 90
- Showcase award: project submission rate >= 90
- Facilitator recommendation marker: all progression thresholds met

## Certificate Triggers

- Term certificate: progression thresholds met for term scope
- Level completion certificate: all term thresholds met and capstone submitted
